# MiniHack: AI Data Intelligence Challenge

## Project Overview
This project analyzes a last-mile delivery dataset using Python, Pandas, and Matplotlib.

## Features
- Data loading and exploration
- Data cleaning and preprocessing
- Peak hour delay analysis
- Weather impact analysis
- Rider experience analysis
- Statistical testing (t-test)
- City-wise, monthly, and vehicle-wise visualizations

## Technologies Used
- Python
- Pandas
- Matplotlib
- SciPy

## Files
- analysis.py
- last_mile_delivery_dataset.csv

## Key Findings
- Peak hour deliveries experience higher delays.
- Rain and fog significantly increase delays.
- Medicine orders are most affected during rain.
- Rider experience has no statistically significant impact on delays.
- Better rider allocation during peak hours can improve performance.
